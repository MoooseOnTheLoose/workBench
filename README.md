## Purpose

`_workBench_` is a personal repository for **learning, experimentation, and diagnostics**.

It contains small tools and scripts used to better understand networks, systems, and low-level behavior in a controlled environment.

This repository is **not intended for production use**.

---

## Intended Use

Tools here are meant for:

- education and skill development  
- personal network diagnostics  
- testing in lab, VM, or sandbox environments  

Use only on systems and networks you own or control.

---

## Safety

Some tools demonstrate security-adjacent behavior for learning purposes.

Please read **SAFETY.md** before using anything in this repository.

---
